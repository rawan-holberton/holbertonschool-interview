# Minimum Operations

## Description

This project focuses on solving an optimization problem using Python.

Given a single character **"H"** in a text file, you are only allowed to use two operations:
- Copy All
- Paste

The goal is to determine the **minimum number of operations** required to obtain exactly `n` occurrences of the character **H**.

This problem teaches algorithmic thinking, especially factorization and optimization strategies.
